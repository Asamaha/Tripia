## Development

### Yelp API keys
```sh
- module.exports = {
 consumer_key: 'YOUR-KEY',
 consumer_secret: 'YOUR-KEY',
 token: 'YOUR-KEY',
 token_secret:  'YOUR-KEY',
 ssl: true
};
```
### Installing Dependencies

From within the root directory:

```sh
- npm install
- bower install
```
##### To run the Server

- ``` node server.js```
